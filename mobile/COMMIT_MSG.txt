Mobile v1.26: Discover tab (YouTube trending/music/new-releases, tap-to-download)

New Discover feature (YouTube Data API v3, free tier + 6h cache):
- Discover.kt: trending (mostPopular, 1 quota unit) by region + category, and
  search (newest first), with disk caching for 6h and stale-cache fallback so a
  shared/bundled key's daily quota lasts. Key comes from BuildConfig (bundled via
  secret.properties → YOUTUBE_API_KEY) or an in-app override the user pastes.
- DiscoverScreen: shelves — "Trending in Kenya", "Trending Music", "Trending
  Worldwide", and "New from <your top artist>" — each a horizontal art shelf.
  MP3/MP4 toggle; tap any card to queue the download (one tap). Shows a "Set
  YouTube key" prompt + dialog when no key is configured.
- Reached via a gradient "Discover" banner on Home + a drawer entry (tab 6). Bottom
  nav stays 5 tabs, so Home stays fast.

Note on personalization: YouTube's API does NOT expose personal "Top Picks For You",
so we approximate with trending + "new from your most-played artist".

To enable: paste a free YouTube Data API v3 key in Discover (or set YOUTUBE_API_KEY
in secret.properties before building).
