Mobile v1.32: cleanup + performance — APK 153MB -> 77MB, instant tab switching

Size (safe — no functional change on real devices):
- abiFilters = armeabi-v7a + arm64-v8a only. Dropped x86/x86_64 native libs, which
  are emulator-only and were ~half the engine payload.
- Removed the unused aria2c dependency (we never used --external-downloader), which
  bundled a large native lib.
- Result: APK 153.2MB -> 77.3MB; install size falls proportionally. The Python +
  ffmpeg engine for ARM is untouched, so all downloads/conversions still work.

Performance:
- Removed the Crossfade tab transition — tabs now switch INSTANTLY (no 150ms fade,
  no double-composition of the outgoing + incoming screen). Local-app responsiveness.
- Background work (downloads, AI jobs, playlist analysis) already runs on app-scope
  coroutines, so the UI thread is never blocked by them.

Repo cleanup:
- Removed the committed "IMAGES FORM PHONE" screenshot folder (no app purpose).

Note: the install size being larger than the APK is normal Android behaviour (it
extracts native libs + compiles dex on install); the ABI trim shrinks both.
